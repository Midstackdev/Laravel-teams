## About Laravel teams

Laravel teams management. A cool team management feature app with subscriptions.


## Features

- Laravel 6
- Vue.js
- Stripe
- Laravel Cashier
- Laratrust Package

## Instalation

- Clone the repo
- Copy .env.example to .env
- Configure .env
- Run composer install
- Run php artisan key:generate
- Run php artisan migrate
- Run npm install
